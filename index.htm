How to Install a Bookmarklet

Follow these steps to install a bookmarklet from an <a> element with JavaScript code in the href attribute.

General Steps

  1. Right-click the bookmarklet link and select "Copy Link Address" or copy the javascript: code manually.
  2. Create or edit a bookmark in your browser and paste the copied code into the URL field.
  3. Save the bookmark. It will now work as a bookmarklet when clicked.

Desktop Browsers

Google Chrome

  1. Right-click the bookmarks bar (use Ctrl+Shift+B to show it if hidden).
  2. Select "Add Page", give it a name, and paste the javascript: code into the URL field.
  3. Click Save.

Mozilla Firefox

  1. Right-click on the bookmarks bar and select "New Bookmark".
  2. Give it a name and paste the javascript: code into the Location field.
  3. Click Add.

Microsoft Edge

  1. Right-click on the bookmarks bar (show it using Ctrl+Shift+B).
  2. Select "Add a Favorite", enter a name, and paste the javascript: code into the URL field.
  3. Click Save.

Safari (Mac)

  1. Show the bookmarks bar via View > Show Favorites Bar.
  2. Drag the <a> link directly to the bar, or manually copy the javascript: code and add it to a new bookmark.

Mobile Browsers

Chrome (Android/iOS)

  1. Bookmark any page.
  2. Edit the bookmark and replace the URL with the javascript: code.
  3. Save the changes.

Safari (iOS)

  1. Bookmark any page (e.g., https://example.com).
  2. Edit the bookmark and replace the URL with the javascript: code.
  3. Save the changes.

Samsung Internet (Android)

  1. Bookmark any page.
  2. Edit the bookmark and replace the URL with the javascript: code.
  3. Save the changes.

Firefox (Android/iOS)

  1. Bookmark any page.
  2. Edit the bookmark and replace the URL with the javascript: code.
  3. Save the changes.

Example Bookmarklet

Here's an example of a bookmarklet link:

<a href="javascript:(function(){alert('Hello, World!');})();">My Bookmarklet</a>

Right-click the link, copy the URL, and follow the instructions above to install it as a bookmarklet.
